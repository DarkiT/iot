IOT 物联网开发 (基于Swoole扩展)
==============
1.概述
--------------
+ 基于tp5.1 框架
+ 根据实际项目开发，用于工作中
+ web后台 +socket服务器 +硬件端数据处理
+ 后台实时刷新 硬件端的状态
+ 本项目借鉴了开源项目 donkey 并加以修改
+ 请使用swoole扩展2.1.0+
